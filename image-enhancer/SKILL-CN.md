# 图像增强器 (Image Enhancer)

此技能获取你的图像和屏幕截图并使它们看起来更好——更清晰、更明晰、更专业。

## 何时使用此技能

- 提高博客文章或文档的屏幕截图质量
- 在社交媒体上分享之前增强图像
- 为演示文稿或报告准备图像
- 放大低分辨率图像
- 锐化模糊照片
- 清理压缩图像

## 此技能做什么

1. **分析图像质量**：检查分辨率、清晰度和压缩伪影
2. **增强分辨率**：智能放大图像
3. **提高清晰度**：增强边缘和细节
4. **减少伪影**：清理压缩伪影和噪点
5. **针对用例优化**：根据预期用途（网络、打印、社交媒体）进行调整

## 如何使用

### 基本增强

```
Improve the image quality of screenshot.png
(提高 screenshot.png 的图像质量)
```

```
Enhance all images in this folder
(增强此文件夹中的所有图像)
```

### 特定改进

```
Upscale this image to 4K resolution
(将此图像放大到 4K 分辨率)
```

```
Sharpen this blurry screenshot
(锐化此模糊的屏幕截图)
```

```
Reduce compression artifacts in this image
(减少此图像中的压缩伪影)
```

### 批量处理

```
Improve the quality of all PNG files in this directory
(提高此目录中所有 PNG 文件的质量)
```

## 提示

- 始终保留原始文件作为备份
- 最适合屏幕截图和数字图像
- 可以批量处理整个文件夹
- 如果需要，指定输出格式（PNG 用于质量，JPG 用于较小尺寸）
- 对于社交媒体，提及平台以获得最佳尺寸

## 常见用例

- **博客文章**：在发布前增强屏幕截图
- **文档**：使 UI 屏幕截图清晰
- **社交媒体**：优化 Twitter, LinkedIn, Instagram 的图像
- **演示文稿**：为大屏幕放大图像
- **打印材料**：提高物理媒体的分辨率
